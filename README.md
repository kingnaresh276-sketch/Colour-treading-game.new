# Color Trading Game Full Mobile-Ready

## Deployment

### Mobile (Termux)
1. Install Termux
2. Install Node.js: `pkg install nodejs`
3. Navigate to folder: `cd /path/to/game`
4. Install dependencies: `npm install`
5. Start server: `node server.js`
6. Open browser: http://localhost:3000
7. Play: login → deposit → bet → withdraw

### Online Hosting (Render.com)
1. Upload folder to GitHub
2. Connect repository in Render.com → Web Service
3. Environment: Node.js
4. Build: `npm install`
5. Start: `npm start`
6. Use the online URL in your browser
7.
